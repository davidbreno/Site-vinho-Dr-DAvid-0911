# Dental Platform Dashboard

Projeto React + Vite produzido para um dashboard odontológico.

Como rodar localmente

1. Instalar dependências:

```powershell
npm install
```

2. Rodar em dev:

```powershell
npm run dev
```

3. Checagem de tipos:

```powershell
npm run typecheck
```

Notas
- Este repositório foi preparado para uso com Vite + SWC e Tailwind (CSS já incluído).
- Recomendo adicionar ESLint, Prettier e testes contínuos em CI.
